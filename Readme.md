# Desafio Técnico – Processo Seletivo IN Júnior

Este repositório contém a resolução do desafio técnico em JavaScript proposto no processo seletivo da IN Júnior.

Os exercícios foram desenvolvidos utilizando JavaScript puro, com foco em lógica de programação, clareza do código, organização e boas práticas.

## Exercícios

Cada arquivo `.js` corresponde à resolução de um exercício do desafio:

1. `exercicio01-bhaskara.js` — Cálculo das raízes de uma equação do segundo grau (fórmula de Bhaskara)
2. `exercicio02-fatorial.js` — Cálculo de fatorial com validação de entrada
3. `exercicio03-idades.js` — Verificação de maioridade com base no ano de nascimento
4. `exercicio04-conversao-moedas.js` — Conversão de valores em reais para dólar e euro
5. `exercicio05-fizzbuzz.js` — Implementação da lógica FizzBuzz
6. `exercicio06-fibonacci.js` — Sequência de Fibonacci utilizando função recursiva
7. `exercicio07-palindromo.js` — Verificação de frase palíndroma
8. `exercicio08-contagem-livros.js` — Contagem de livros por categoria
9. `exercicio09-autores.js` — Listagem de autores sem repetição
10. `exercicio10-fila-atendimento.js` — Simulação de fila de atendimento
11. `exercicio11-ordem-crescente.js` — Verificação de números em ordem crescente
12. `exercicio12-elemento-unico.js` — Identificação de elemento único em um array
13. `exercicio13-gerenciamento-time.js` — Sistema de gerenciamento de jogadores com menu interativo

## Observações

Os códigos foram testados em ambiente de navegador utilizando JavaScript puro.  
Um arquivo `index.html` pode ser utilizado localmente apenas para fins de teste e execução dos scripts.
